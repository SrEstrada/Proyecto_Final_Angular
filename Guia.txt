> cmdkey /list
> cmdkey /delete:git:https://github.com
// git remote -v
> git clone …
> cd Repositorio

> git checkout -b MiRama origin/MiRama

> git checkout -b MiRama

> git push -u origin main ( o TuRama para seguir trabajando con ella)

> git reset --hard
> git pull origin InfoUser
